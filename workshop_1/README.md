## Content Covered
### Basic Folder Structure
```
project_directory
|
-|index.html
-|src
--|imgs
---|your_image.png
--|css
---|style.css
```
### HTML Tags 
* `img` - to add image
* `div` - to organize code
* `h1...h6` - for headings
* `ul` - for bullets
* `ol` - for numbered list
* `li` - for numbered lists and bullets
* `link` - to import `css`
* `body` - to add visible content
### CSS Intro
* CSS is used to add styling to the pages
* A basic `css` file template is provided
